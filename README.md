# promise-polyfill

A Promise polyfill implementation based on the [Promises/A+ specification](https://promisesaplus.com/).

## Installation

```bash
npm install
```

## Running Tests

The project uses the Promises/A+ test suite to validate the implementation.

To run the tests:

```bash
npm run promises-aplus-tests
```

This will execute the test suite and check compliance with the Promises/A+ specification.
